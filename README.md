### ✨ **주요 기능**

- **📅 주간 캘린더**: 한 눈에 보는 주간 캘린더로 원하는 날짜를 빠르게 선택할 수 있습니다.
- **🍚 식당별 메뉴**: 캠퍼스 내 3개의 식당별 오늘의 메뉴를 바로 확인할 수 있습니다.
- **✅ 타임존 보정**: 사용자의 위치나 시간에 상관없이, 항상 정확한 **한국 시간**을 기준으로 메뉴를 보여줍니다.
